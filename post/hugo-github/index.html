새 프로젝트 생성#

hugo new site Homepage

hugo.toml 설정#

cd Homepage

vi hugo.toml

baseURL = 'https://kimnux12.github.io/'
languageCode = 'en-us'
title = 'Linux에 관한 주저리 주러지'
theme = 'hugo-PaperMod'

선택한 테마를 설치하고 git 서브모듈 등록하기#

cd ~/Homepage

git init

git submodule add https://github.com/adityatelange/hugo-PaperMod.git themes/hugo-PaperMod

새 포스트 작성하기#

hugo new post/test.md

hugo 서버 실행#

hogo serve

-D 옵션은 draft:true로 설정되어 있을 경우를 대비하기 위함.

로컬 접속#

localhost:1313 으로 접속해서 상태 확인

https://github.com/kimnux12/kimnux12.github.io 설정 확인 :ATTACH:#

Settings -> Pages -> Select the "master" branch -> Save

’Deploy from a branch’ 선택 -> master 선택 -> save 출처 : https://dilipsundarraj.com/own-website/hugo-github-pages/

깃허브 원격저장소에 hugo website 올리기#

빌드하기#

cd ~/Homepage

hugo

kimnux12.github.io를 로컬 저장소로 클론하기#

git clone https://kimnux12:token@github.com/kimnux12/kimnux12.github.io.git

public 폴더의 내용을 clone 받은 폴더로 옮기기#

mv ~/Homepage/public/* ./kimnux12.github.io

kimnux12.github.io 원격저장소로 push하기#

git add .

git commit -am 'adding all the content'

git push

추후 새로운 글을 작성하면, 빌드하고 ./kimnux12.github.io로 복사(copy)하고 push하면 된다.
